## üöÄ Setup & Deploy

1. **Place Your PNG Files**  
   Drop the five PNGs in the root of the repo with these exact names and order.

2. **Commit & Push to main**  
   ```bash
   git add .
   git commit -m "Add PNG icons and pages"
   git push origin main
   ```

3. **GitHub Pages**  
   - Go to Settings > Pages.
   - Source: Deploy from branch (the workflow creates the deployment branch automatically).
   - Visit your site at `https://traceybirk.github.io/image-hosting/`

4. **Automatic Deployment**  
   Every push to main updates the live site, thanks to the included GitHub Actions workflow.
